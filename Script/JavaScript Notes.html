JavaScript Notes

JavaScript (JS) is a programming language that adds interactivity to web pages.

Variables: var, let, const.

Data Types: String, Number, Boolean, Array, Object, Null, Undefined.

Functions: function myFunction() {} or const myFunction = () => {}.

DOM Manipulation: document.querySelector(), document.getElementById(), document.createElement().

Events: onclick, onmouseover, addEventListener().

Loops & Conditions: for, while, if-else, switch.

ES6 Features: Arrow functions, template literals, destructuring, spread/rest operators.

APIs & Async JS: fetch(), async/await, JSON.
